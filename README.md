exploratory data analysis for telehealth


problem statement
Patient satisfaction is a critical measure of healthcare quality, especially in telehealth where interactions occur remotely. However, healthcare providers often lack clear insights into which factors most influence patient satisfaction during virtual visits.

This project aims to analyze telehealth visit data to identify how operational factors (e.g., wait time, visit duration), technological conditions (e.g., internet quality, technical issues, device used), visit characteristics (e.g., provider specialty, visit type), and patient demographics affect satisfaction scores and the likelihood that patients would recommend the service.

Understanding these factors is essential for improving the telehealth experience, increasing patient loyalty, and supporting data-driven decisions that enhance service quality.
